### Bonus Bet Conversion: The Math

Every time you receive bonus bets, convert them to cash using this process:

#### The Conversion Formula

Bonus bets are **"stake not returned"** — you only keep the profit, not the original stake. This means you need to hedge.

```
Optimal conversion odds range: +300 to +500 (underdog side)
Typical conversion rate: 70-80% of bonus bet face value

Example: Converting a $100 bonus bet

1. Place $100 BONUS BET on Underdog at +400 odds (Book A)
   If wins: $400 profit (stake not returned)

2. Place $308 CASH HEDGE on Favorite at -400 odds (Book B)
   If wins: $77 profit

Outcomes:
- Underdog wins: +$400 (bonus) - $308 (hedge lost) = +$92 profit
- Favorite wins: -$0 (bonus lost, it was free) + $77 (hedge profit) = +$77 profit

Guaranteed profit: $77-$92 regardless of outcome
Conversion rate: 77-92% of $100 = excellent
```

**Rule: Higher underdog odds = better conversion, but harder to find matching hedge lines.**

### Key Model Features for Bonus Optimization

```python
# bonus_optimizer.py — NEW module

def rank_games_for_bonus_conversion(predictions, odds_data):
    """
    For each upcoming game, calculate:
    1. Max odds spread across books (wider = better conversion)
    2. Best available underdog odds in +300-500 range
    3. Model confidence (higher = safer for unhedged bets)
    4. Book-specific line vs model line (biggest disagreement = most value)
    """
    rankings = []
    for game in predictions:
        # Find widest odds gap across all books for this game
        all_odds = odds_data[game['game_id']]
        max_spread = max(all_odds) - min(all_odds)

        # Find best underdog odds in conversion-optimal range
        underdog_odds = [o for o in all_odds if 300 <= o <= 500]

        # Model confidence = distance from 50%
        confidence = abs(game['model_win_prob'] - 0.5)

        rankings.append({
            'game': game,
            'conversion_score': max_spread * 10 + len(underdog_odds) * 5,
            'confidence_score': confidence,
            'best_conversion_odds': max(underdog_odds) if underdog_odds else None,
        })

    return sorted(rankings, key=lambda x: x['conversion_score'], reverse=True)
```

### Kelly Criterion for Bonus Bets vs Real Money

**For bonus bets (house money):** Use FULL Kelly or even 1.5x Kelly. The downside is capped at $0 since it's free money.

```python
def kelly_fraction(prob, decimal_odds, is_bonus=False):
    """
    prob: your estimated win probability
    decimal_odds: payout ratio (e.g., 3.0 for +200)
    is_bonus: True if using bonus bet (no stake return)
    """
    b = decimal_odds - 1  # net odds
    q = 1 - prob

    kelly = (b * prob - q) / b

    if is_bonus:
        return min(kelly * 1.5, 1.0)  # aggressive with house money, cap at 100%
    else:
        return min(kelly * 0.25, 0.05)  # quarter Kelly, max 5% for real money
```

**For real money (post-bonus):** Use quarter Kelly, max 5% of bankroll per bet.

```
Example: Model says Team A has 60% chance to win, odds are +150 (decimal 2.5)

b = 1.5, p = 0.60, q = 0.40
Kelly = (1.5 * 0.60 - 0.40) / 1.5 = (0.90 - 0.40) / 1.5 = 0.333

Quarter Kelly = 0.333 * 0.25 = 0.083 = 8.3%
Capped at 5% = bet 5% of bankroll

On $1,000 bankroll: bet $50
```

### The 5% Hard Cap

**Never bet more than 5% of your REAL MONEY bankroll on a single bet.** This is non-negotiable. Even if the model shows a 15% edge, variance can destroy you.

The only exception: bonus bets (house money) can be sized larger since the downside is $0.

## APPENDIX A: Quick Reference — Hedging Formulas

### Formula 1: Equal-Profit Hedge (for insurance/risk-free bets)

```
Hedge_Amount = (Original_Bet × Original_Decimal_Odds) / (1 + Hedge_Decimal_Odds)
```

### Formula 2: Bonus Bet Conversion Hedge

```
Hedge_Amount = (Bonus_Amount × (Bonus_Decimal_Odds - 1)) / Hedge_Decimal_Odds
```

### Formula 3: Implied Probability from American Odds

```python
def american_to_implied(odds):
    if odds > 0:
        return 100 / (odds + 100)
    else:
        return abs(odds) / (abs(odds) + 100)

# Examples:
# -110 → 52.38%
# +200 → 33.33%
# -300 → 75.00%
# +150 → 40.00%
```

### Formula 4: Edge Calculation

```
Edge = Model_Probability - Market_Implied_Probability

Only bet when Edge > 0.03 (3%)
```

### Formula 5: Conversion Rate Estimator

```
Conversion_Rate ≈ (Bonus_Decimal_Odds - 1) / Bonus_Decimal_Odds

+300 (4.0 decimal): (4.0 - 1) / 4.0 = 75%
+400 (5.0 decimal): (5.0 - 1) / 5.0 = 80%
+500 (6.0 decimal): (6.0 - 1) / 6.0 = 83%
```
